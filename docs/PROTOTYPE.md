# 프로젝트 개요

- 프로젝트 명 : semi 페이스북
- 기획 의도/목표 : CRUD 구현 및 스프링(부트) 공부
- 가상 타겟 : 친구와 교류하고 싶은 사람들
- 기대 효과 : 친목 활성화

# 요구사항 명세

- 사용자 관점(User Story)
    - 회원으로서 가입할 수 있다
    - 친구와 방명록을 주고받을 수 있다
    - 자신의 기분을 표현할 수 있다

- 기능 요구사항
    - 회원 관리
        - 회원가입(이메일 인증) 및 로그인
        - 유저 정보 변경(이메일, 비밀번호 등)
        - 상태 메세지 표현
    - 게시판
        - 방명록 작성
        - 방명록 조회
        - 작성된 방명록에 댓글
    - 친구 관리
        - 친구 요청 - 수락 및 거절
        - 친구 목록 관리

- 비기능 요구사항
    - 보안 : 회원 비밀번호는 암호화하여 관리
    - 성능 : 주요 api는 500ms 이내 응답
